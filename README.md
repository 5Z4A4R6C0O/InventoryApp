Inventory Management System (Professional Suite)

Este sistema es una solución de escritorio robusta diseñada para la gestión de inventarios y control de existencias. El enfoque principal del desarrollo fue la escalabilidad y la integridad de los datos, utilizando un paradigma de Programación Orientada a Objetos (POO) para facilitar el mantenimiento y la expansión de funcionalidades.

Características Principales (Core Features)

Módulo de Mapeo de Sinónimos: Implementación de un sistema inteligente de categorización que normaliza la entrada de datos. El sistema vincula automáticamente términos relacionados (ej. "Teclado" → "Tecnología") mediante diccionarios de mapeo personalizados.

Gestión de Datos mediante POO: Arquitectura basada en clases y herencia para manejar entidades de productos, usuarios y transacciones de forma aislada y segura.

Persistencia de Datos: Manejo de archivos locales para el almacenamiento de registros, garantizando la recuperación de información entre sesiones.

Búsqueda Optimizada: Implementación de algoritmos de filtrado eficientes para la localización rápida de artículos dentro de grandes volúmenes de datos.

Interfaz de Usuario (GUI): Experiencia de usuario intuitiva desarrollada con Tkinter, enfocada en la eficiencia operativa.

Stack Tecnológico

Lenguaje: Python 3.x

Librerías: Tkinter (Interfaz), JSON/CSV (Persistencia), OS/Sys (Gestión de rutas).

Retos Técnicos Resueltos

Categorización Automática: El mayor reto fue diseñar una estructura que permitiera al sistema "aprender" o reconocer que diferentes términos pertenecen a una misma categoría sin necesidad de intervención manual constante.

Modularidad: Se logró separar completamente la lógica del backend (cálculos y gestión de archivos) de la interfaz visual, permitiendo que en un futuro se pueda migrar a una interfaz web sin reescribir la lógica base.
